# Noorwegen2026

One page web app voor reis door noorwegen.

Trip content: `trip-data.json`. Agent context: `CLAUDE.md`.

## Trip suggestions (phone → site)

1. **iOS Shortcut** → GitHub Issue — install `shortcuts/Noorwegen-tip.shortcut` — [docs/ios-shortcut-setup.md](docs/ios-shortcut-setup.md)
2. **Claude Code** processes issues with `/process-trip-suggestion` (skill in `.claude/skills/`)
