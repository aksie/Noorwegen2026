# Noorwegen tip shortcut

Pre-built iOS Shortcut that creates a GitHub Issue for trip suggestions.

## Install (iPhone)

1. Get `Noorwegen-tip.shortcut` on your phone (AirDrop from Mac, iCloud Drive, or download from GitHub).
2. Tap the file → **Add Shortcut**.
3. Open the shortcut in the Shortcuts app → edit the **first Text** action → paste your fine-grained GitHub token (`github_pat_…`).
4. Tap **ⓘ** → enable **Show in Share Sheet** → types: **Text**.

Token setup and labels: [docs/ios-shortcut-setup.md](../docs/ios-shortcut-setup.md)
